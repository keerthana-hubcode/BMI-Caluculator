BMI CALCULATOR PROJECT
=====================

Project Name:
BMI Calculator

Description:
The BMI (Body Mass Index) Calculator is a simple web-based application that allows users to calculate their BMI based on height and weight. The project helps users understand their health category such as Underweight, Normal weight, Overweight, or Obese using standard BMI ranges.

This project is developed using basic web technologies and is suitable for beginners to understand form handling, calculations, and dynamic result display in JavaScript.

Technologies Used:
- HTML
- CSS
- JavaScript

Features:
- User-friendly interface
- Takes height and weight as input
- Calculates BMI instantly
- Displays BMI value with health category
- Input validation for better accuracy
- Responsive design for different screen sizes

How to Use:
1. Open the index.html file in any web browser.
2. Enter your height and weight in the given input fields.
3. Click on the "Calculate BMI" button.
4. View your BMI result and health status on the screen.

BMI Categories:
- Underweight: BMI less than 18.5
- Normal weight: BMI between 18.5 and 24.9
- Overweight: BMI between 25 and 29.9
- Obese: BMI 30 or above

Project Structure:
- index.html  : Contains the structure of the application
- style.css   : Handles styling and layout
- script.js   : Contains BMI calculation logic

Future Enhancements:
- Add age and gender-based BMI analysis
- Display health tips based on BMI result
- Store BMI history using local storage
- Improve UI with animations
